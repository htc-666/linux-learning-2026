# 第26章 串口应用编程 ⭐⭐⭐(嵌入式必备,记套路不记参数)

> 串口概念(波特率/数据位/停止位/校验)= Windows 用过,跳。只学 Linux 怎么操作串口。
> 核心: 串口=设备文件 + termios 配置。那堆标志位不用背,用 cfmakeraw + 抄模板。

---

## 一、主线套路(背这个流程就行)

```
① open("/dev/ttyUSB0", O_RDWR|O_NOCTTY)    串口=设备文件,当文件打开
② tcgetattr(fd, &opt)                       读出当前配置到 termios
③ 改 opt:
     cfmakeraw(&opt)                         ⭐一键设成 raw 原始模式
     cfsetispeed/cfsetospeed(&opt, B115200)  设波特率
④ tcsetattr(fd, TCSANOW, &opt)              写回,配置生效
⑤ read(fd,...) / write(fd,...)              收发数据(跟文件IO一样)
⑥ close(fd)
```

---

## 二、关键概念(必记)

### 1. 串口 = 设备文件
```
/dev/ttySx   板载串口
/dev/ttyUSBx USB转串口模块
→ open/read/write/close 操作,和文件IO一模一样(比Windows的CreateFile还直观)
```

### 2. termios 配置三步
```
tcgetattr(读出当前配置) → 改 termios → tcsetattr(写回生效)
波特率必须用专门函数 cfsetispeed/cfsetospeed,不能直接赋值
```

### 3. 两种工作模式 ⭐(Linux 的坑,核心理解)
```
规范模式 canonical(默认) = "带秘书的"
  - 按【行】处理,要等回车 read() 才返回
  - 处理特殊键(退格/Ctrl+C/Ctrl+D)、回显
  → 适合【人敲命令行】

原始模式 raw = "不要秘书"
  - 逐字节,来一个 read() 立刻返回,不等回车
  - 不解释特殊字符(0x03就是数据,不当Ctrl+C)、不回显
  → 适合【机器间传数据】(原样逐字节)
```

### 4. 为什么传数据必须用 raw(关键)
```
串口收 传感器/模块 数据:
  规范模式: 数据没回车 → read()傻等不返回; 数据含0x03 → 被当Ctrl+C搞乱
  raw模式:  来啥给啥,不等回车、不乱解释 → 数据完整准确
→ 只要"传数据"(传感器/模块/二进制),一律 raw
→ 规范模式给"人敲命令"用,raw给"机器传数据"用
```

### 5. 设 raw 用 cfmakeraw,不用手配标志位
```c
struct termios opt;
tcgetattr(fd, &opt);
cfmakeraw(&opt);                  // ⭐一行设好raw(内部把c_cflag那堆标志全配了)
cfsetispeed(&opt, B115200);
cfsetospeed(&opt, B115200);
tcsetattr(fd, TCSANOW, &opt);
```
> 那堆 c_cflag/c_iflag/c_oflag/c_lflag 标志位**不用背**——cfmakeraw 替你设了,或抄模板改波特率即可。

## 三、完整模板(用时抄,改设备名+波特率)

```c
#include <fcntl.h>
#include <termios.h>
#include <unistd.h>

int fd = open("/dev/ttyUSB0", O_RDWR | O_NOCTTY);   // O_NOCTTY:不作为控制终端
if (fd < 0) { perror("open"); return -1; }

struct termios opt;
tcgetattr(fd, &opt);            // 读当前配置
cfmakeraw(&opt);                // 设raw模式
cfsetispeed(&opt, B115200);     // 输入波特率
cfsetospeed(&opt, B115200);     // 输出波特率
tcsetattr(fd, TCSANOW, &opt);   // TCSANOW=立即生效

write(fd, "AT\r\n", 4);         // 发数据
char buf[128];
int n = read(fd, buf, sizeof(buf));   // 收数据
close(fd);
```
